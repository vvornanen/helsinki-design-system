Placeholders

Placeholders can be used to replace missing content or denote areas left intentionally blank.

Examples of possible use cases for placeholders:

  • In place of a missing image for example in a search listing blog posts, news items or events.
  • As a default image preview for an user uploaded image.

Usage

Default HDS placeholders include image placeholders and user image placeholders for different sizes.

  • Always use the image size closest to your needs. The size of the placeholder icon has been optimised to the image size.
  • Do not scale images larger. If you need a size between available sizes, choose the next largest image file.
  • You can crop the images to suit your aspect ratio.
